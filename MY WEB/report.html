introduction:

I based my site on "Feudal Japan"/ Medieval Japan as I believe there is a niche community that truly are passionate about this. Also, I personally truly love history and Medievel Japan is a very intriguing part of world history that i enjoy.

Background:

My target audience are mainly budding historians, and individuals that enjoy learning about Medievel Japan and its culture. For my website I found many inpiration for it, mainly in Japanese art. This is what determined my background theme as I found it to give off an authentic medievel japanese feel to my website.

Also, a website that inspired me was https://colorlib.com/wp/community-website-templates/. I really like how the landing page is very simple and not extremely detailed, easy for the user to navigate. Another inspiration for my website was the Apple Airpod pro websites https://www.apple.com/uk/airpods-pro/. It had a scroll feature where if the user scrolls the mouse a video plays in the background, althogh i didn't emmulate that I was intrigued to add a scroll feature. I also liked how interactive the apple website was, so I wanted to delve into making a website where elements would show upon scroll of the page.

Planning:

I decided that for my website a hierarchical structure would be best, as it is simple and easy for the user to navigate. My website is easy to view, with apropriate colours, making sure that text is easy to read as. I ahave insured by making my white font has a low opacity black bakcground to it. Also, I have used alt tags to describe all my images in case it does not show up, so it will not hinder the users experience. My website is also responsive, able to view on multiple viewports

Implementation:

As I had mentioned before, I wanted to implement a scroll feature, where elemts appear upon scroll. This was a very difficult process as I found it difficult to develop Javascript. Hence, I went to StackOverflow and found code where it shows elements after certain number of pixels, I decided to implement it in my website. I modified this code slightly to adjust to my website. I also learnt how to correctly adapt this JQuery to the different viewports so the website is functioniong correctly with the code. I also learnt about how important is to positioning elements correctly in order for it to be displayed correctly on screen. there were many videos I used on youtube to understand these concepts. Additonally, I learned how to correct;y format images in my website so when the window size is changed the images will resize accordingly. I also learned from a yutube video on how to make a hamburger navigation bar, which only displays at certain viewports.
